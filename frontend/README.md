# Frontend Setup:

## Requirements

Ensure each of the following are downloaded/installed prior to running the project:

- Git
- VS Code (Our selected IDE)
- Node.js
- Expo Go App for Mobile Testing
- VS Code Extensions such as Prettier and Eslint 

## Running the frontend

Follow these instructions to run the frontend section of the project:

1. Clone the project repo

   ```bash
   git clone <repo-url>
   ```

2. Go to "frontend" directory

   ```bash
   cd frontend
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the app

   ```bash
   npx expo start
   ```

## Running the app on mobile devices

- Ensure "Expo Go" is installed on your phone
- Scan the QR code found in the terminal after running the "npx expo start" command

## Running the app on web browser

- After running the "npx expo start" command, enter:

  ```bash
  w
  ```

- Which will open the app on the web browser
